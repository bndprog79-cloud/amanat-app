⚡ САМЫЙ БЫСТРЫЙ СПОСОБ (Codemagic - 15 минут):
═════════════════════════════════════════════════════════════════════════════

ШАГ 1: GitHub (5 минут)
  1. Создать репозиторий: https://github.com/new
  2. Имя: amanat-app (Public)
  3. Загрузить код через Git

ШАГ 2: Codemagic (2 минуты)
  1. Перейти: https://codemagic.io
  2. Sign up with GitHub
  3. Выбрать репозиторий amanat-app
  4. Set up build → Android → Debug

ШАГ 3: Сборка (5-7 минут)
  1. Start your first build
  2. Ждать завершения

ШАГ 4: Скачивание (1 минута)
  1. Download APK

ШАГ 5: Установка (2 минуты)
  1. Загрузить на телефон
  2. Установить APK
