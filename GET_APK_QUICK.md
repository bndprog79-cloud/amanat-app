## 🥇 СПОСОБ 1: Codemagic (ЛУЧШИЙ - рекомендуется)

```
⏱️ Время: 10 минут
💰 Стоимость: Бесплатно
⭐ Сложность: Очень простая
```

### Пошаговая инструкция:

**Шаг 1: Регистрация (2 мин)**
```
1. Открыть https://codemagic.io
2. Нажать "Sign up with GitHub"
3. Разрешить доступ к репозиториям
```

**Шаг 2: Добавить проект (1 мин)**
```
1. Нажать "Connect repository"
2. Выбрать "amanat-app" (или "New repository")
3. Нажать "Set up build"
```

**Шаг 3: Выбрать платформу (30 сек)**
```
1. Платформа: Android
2. Тип: Debug
3. Нажать "Save"
```

**Шаг 4: Собрать (5 мин)**
```
1. Нажать "Start your first build"
2. Смотреть прогресс (сборка займет 5-7 минут)
3. Когда ✅ - готово!
```

**Шаг 5: Скачать (30 сек)**
```
1. Нажать на завершенную сборку
2. Раздел "Artifacts"
3. Скачать "amanat-app-debug.apk"
```

### ✅ Готово! APK в папке Downloads

## 🥈 СПОСОБ 2: GitHub Actions (АВТОМАТИЧЕСКИЙ)

```
⏱️ Время: 10 минут (+ автосборка каждый push)
💰 Стоимость: Бесплатно
⭐ Сложность: Простая
```

### Пошаговая инструкция:

**Шаг 1: Загрузить на GitHub (3 мин)**
```bash
cd amanat_app
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/amanat-app.git
git push -u origin main
```

**Шаг 2: Создать папку для workflow (1 мин)**
```
Создать локально:
  amanat_app/
  └── .github/
      └── workflows/
          └── build_apk.yml
```

**Шаг 3: Скопировать конфигурацию (1 мин)**
```
Скопировать содержимое файла build_apk.yml
В созданный файл .github/workflows/build_apk.yml
```

**Шаг 4: Загрузить workflow (1 мин)**
```bash
git add .github/
git commit -m "Add build workflow"
git push
```

**Шаг 5: Запустить сборку (5 мин)**
```
1. Открыть https://github.com/YOUR_USERNAME/amanat-app
2. Вкладка "Actions"
3. "Build APK" → "Run workflow"
4. Выбрать "main" → "Run workflow"
5. Ждать 5-7 минут
```

**Шаг 6: Скачать (30 сек)**
```
1. Нажать на завершенную сборку
2. Раздел "Artifacts"
3. Скачать "app-debug"
```

### ✅ Готово! Плюс: автосборка при каждом push

## 📞 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Codemagic не компилирует?
```
1. Проверить, что репозиторий публичный (Public)
2. Проверить, что в репозитории есть папка app/
3. Нажать "Clear build cache" и пересобрать
```

### GitHub Actions?
```
1. Проверить, что загружена папка .github/workflows/
2. Проверить синтаксис build_apk.yml
3. В Actions нажать "Re-run all jobs"
```

### AppCenter?
```
1. Убедиться, что выбрана правильная ветка (main)
2. Проверить, что репозиторий подключен
3. Нажать кнопку Build еще раз
```
