## ШАГ 1️⃣: СОЗДАТЬ РЕПОЗИТОРИЙ НА GITHUB (5 минут)

### 1.1 Создать репозиторий

```
1. Открыть https://github.com/new
   
2. Ввести имя репозитория:
   ┌─────────────────────────────────┐
   │ amanat-app                      │
   └─────────────────────────────────┘

3. Выбрать:
   ☑️ Public (публичный)
   
4. Нажать:
   [Create repository]
```

### 1.2 Загрузить код

**На компьютере, в терминале:**

```bash
# Перейти в папку проекта
cd ~/amanat_app

# Инициализировать Git
git init

# Добавить все файлы
git add .

# Создать первый коммит
git commit -m "Initial commit: AMANAT app"

# Переименовать основную ветку
git branch -M main

# Добавить удаленный репозиторий
git remote add origin https://github.com/YOUR_USERNAME/amanat-app.git

# Загрузить код на GitHub
git push -u origin main
```

⚠️ **Замените YOUR_USERNAME на ваше имя пользователя GitHub!**

### 1.3 Проверить загрузку

```
1. Открыть https://github.com/YOUR_USERNAME/amanat-app
2. Должны видеть все файлы проекта
3. ✅ Репозиторий готов!
```

## ШАГ 2️⃣: РЕГИСТРАЦИЯ НА CODEMAGIC (2 минуты)

### 2.1 Открыть Codemagic

```
1. Открыть в браузере:
   https://codemagic.io
   
2. Должны увидеть:
   ┌─────────────────────────────────┐
   │  CODEMAGIC                      │
   │                                 │
   │  [Sign up with GitHub]          │
   │  [Sign in with GitLab]          │
   │  [Sign in with Bitbucket]       │
   └─────────────────────────────────┘
```

### 2.2 Зарегистрироваться через GitHub

```
1. Нажать: [Sign up with GitHub]

2. На странице GitHub:
   ┌─────────────────────────────────┐
   │ Authorize codemagic             │
   │                                 │
   │ [Authorize codemagic/codemagic] │
   └─────────────────────────────────┘

3. Нажать: [Authorize codemagic/codemagic]

4. GitHub попросит пароль - введите пароль

5. Вернетесь на Codemagic
```

### 2.3 Разрешить доступ к репозиториям

```
1. Codemagic попросит доступ к своим репозиториям
   
2. Выбрать:
   ☑️ Only select repositories
   
3. Выбрать: amanat-app
   
4. Нажать: [Save]
```

✅ **Готово! Зарегистрированы на Codemagic**

## ШАГ 6️⃣: СКАЧАТЬ APK (1 минута)

### 6.1 Скачать файл

```
1. На странице с результатом сборки
2. Нажать кнопку: [Download]
3. Браузер предложит сохранить файл
4. Выбрать папку: Downloads
5. Нажать: [Save]
```

### 6.2 Проверить файл

```
После скачивания:

📁 Downloads/
   └─ amanat-app-debug.apk (47 МБ)

✅ APK готов к установке!
```

## ШАГ 7️⃣: УСТАНОВИТЬ НА ТЕЛЕФОН (5 минут)

### 7.1 Загрузить на телефон (способ 1: облако)

```
1. Открыть Google Drive:
   https://drive.google.com

2. Нажать: [+ New] → [File upload]

3. Выбрать: amanat-app-debug.apk

4. Дождаться загрузки (2-3 минуты)

5. На телефоне:
   - Открыть Google Drive
   - Найти amanat-app-debug.apk
   - Нажать на файл
   - Выбрать: [Download]
```

### 7.2 Установить приложение

```
На телефоне:

1. Открыть файловый менеджер
   
2. Перейти в: Downloads

3. Найти: amanat-app-debug.apk

4. Нажать на файл
   
5. Должно спросить:
   "Разрешить установку из неизвестных источников?"
   → Нажать: [РАЗРЕШИТЬ]

6. Нажать: [УСТАНОВИТЬ]

7. Дождаться завершения (30-60 секунд)

8. Нажать: [ОТКРЫТЬ]

✅ Приложение установлено и готово к использованию!
```

## 💡 СОВЕТЫ

### Пересобрать APK (если нужны изменения)

```
1. Внести изменения в код на компьютере
2. Загрузить на GitHub:
   git add .
   git commit -m "Describe changes"
   git push

3. На Codemagic:
   - Открыть проект
   - Нажать "Build" еще раз
   - Дождаться (5 минут)
   - Скачать новый APK
```

### Повторная установка

```
1. На телефоне удалить старую версию:
   Параметры → Приложения → AMANAT → [Удалить]

2. Загрузить новый APK
3. Установить по тому же процессу
```

### Если что-то не работает

```
1. Убедиться, что репозиторий Public (публичный)
2. Проверить, что загружена папка app/
3. На Codemagic нажать "Clear build cache"
4. Попробовать собрать еще раз
```
