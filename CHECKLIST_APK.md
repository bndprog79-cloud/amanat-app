## 🐛 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК

### GitHub - код не загружается

```
❌ "fatal: 'origin' does not appear to be a 'git' repository"

✅ Решение:
   1. git remote add origin https://github.com/YOUR_USERNAME/amanat-app.git
   2. git push -u origin main
```

### Codemagic - сборка не запускается

```
❌ "Build failed"

✅ Решение:
   1. Убедитесь, что репозиторий PUBLIC
   2. На Codemagic нажмите "Clear build cache"
   3. Попробуйте собрать еще раз
```

### Телефон - APK не устанавливается

```
❌ "Parse error"

✅ Решение:
   1. Удалите старую версию приложения
   2. Скачайте новый APK
   3. Убедитесь, что "Установка из неизвестных источников" включена
```
